### 4 WEEKS BEFORE

- [ ] Confirm booking and receive signed agreement
- [ ] Confirm participant numbers
- [ ] Confirm venue and site visit if needed
- [ ] Send client welcome email with what to expect
- [ ] Begin experience customisation if required
- [ ] Confirm any special requirements (dietary, accessibility, language)
- [ ] Order or prepare all materials and equipment

### 2 WEEKS BEFORE

- [ ] Confirm final participant numbers with client
- [ ] Send client logistics information pack
- [ ] Confirm facilitator team and briefing schedule
- [ ] Prepare all printed materials
- [ ] Test all technology and equipment
- [ ] Prepare name tags and group allocations if required
- [ ] Confirm parking and venue access arrangements
- [ ] Send client reminder email with day-of details

### 1 WEEK BEFORE

- [ ] Facilitator team briefing completed
- [ ] All materials packed and checked
- [ ] Confirm venue setup time with venue coordinator
- [ ] Confirm catering if applicable
- [ ] Prepare facilitator scripts and debrief questions
- [ ] Prepare timing guide
- [ ] Confirm emergency contact numbers
- [ ] Check weather forecast if outdoor activity

### DAY BEFORE

- [ ] Final materials check
- [ ] Pack all equipment in correct order for setup
- [ ] Confirm start time with client
- [ ] Charge all devices and technology
- [ ] Print all participant materials
- [ ] Confirm facilitator team availability and arrival time
- [ ] Prepare facilitator briefing notes

### DAY OF EVENT — SETUP (2 hours before start)

- [ ] Arrive at venue minimum 90 minutes before participants
- [ ] Complete venue setup according to experience requirements
- [ ] Test all technology and audio visual equipment
- [ ] Set out all materials and equipment
- [ ] Brief support facilitators
- [ ] Confirm catering arrangements with venue
- [ ] Do a walk-through of the full experience
- [ ] Welcome client contact on arrival

### DAY OF EVENT — DURING

- [ ] Welcome participants and introduce Team Connect
- [ ] Deliver opening brief and rules
- [ ] Monitor energy and adjust pacing as needed
- [ ] Manage timing against schedule
- [ ] Address any issues or concerns immediately
- [ ] Photograph key moments (with permission)
- [ ] Complete the experience and transition to debrief

### DAY OF EVENT — DEBRIEF AND CLOSE

- [ ] Deliver structured debrief using prepared questions
- [ ] Connect experience to workplace behaviour
- [ ] Distribute any take-away materials
- [ ] Thank client and participants
- [ ] Collect feedback forms if applicable
- [ ] Pack down and leave venue clean

### POST EVENT

- [ ] Send client thank you email within 24 hours
- [ ] Send post-event summary and insights to client
- [ ] Invoice client
- [ ] File all event documentation
- [ ] Team debrief — what worked and what to improve
- [ ] Update experience notes for future reference

***

### MATERIALS AND EQUIPMENT CHECKLIST

**Fynbos sensory walk station:**
- [ ] Collected fynbos specimens (or essential oil substitutes if indoor)
- [ ] Sensory prompt cards — laminated, placed along walk path or at sensory table
- [ ] Plant identification cards — one set per station position
- [ ] Guided walk script — for station facilitator

**Stress-mapping station:**
- [ ] Large stress-mapping template posters — one per table (post on wall or lay flat)
- [ ] Coloured markers — multiple colours per table
- [ ] Stress response reference cards — one per table
- [ ] Sticky notes — multiple colours

**Peer-coaching station:**
- [ ] Coaching framework cards (GROW model or equivalent) — one per participant pair
- [ ] Coaching question prompt cards — one set per pair
- [ ] Private reflection journals — one per participant

**Digital detox creative station:**
- [ ] Watercolour paint sets — one per participant
- [ ] Drawing paper — multiple sheets per participant
- [ ] Pencils — one set per participant
- [ ] Participant journals (drawing/writing)
- [ ] Creative prompt cards — one per participant
- [ ] Optional guided audio (ambient nature sounds) — test at comfortable volume

**Breathwork station:**
- [ ] Yoga mats — one per participant (confirm whether client/venue provides or Team Connect provides)
- [ ] Guided breathwork script — for station facilitator (trained breathwork facilitator required)
- [ ] Grounding exercise cards — one near each mat
- [ ] Optional essential oil roller bottles as take-away gifts

**Closing and take-away materials:**
- [ ] Wellbeing commitment cards — one per participant
- [ ] Team resilience charter templates — one per team/table (large format)

**AV and atmosphere:**
- [ ] PA system and microphone (for opening and closing)
- [ ] Soft background music playlist (for transitions and creative station) — test volume
- [ ] Printed session flow guide for each station facilitator

***

- [ ] Know the procedure for severe weather (outdoor events)
